更新内容

欢迎访问 1.0.1 版本的 Dim Aurora’s Blog！本站目前已经完成了基于 Hexo Vivia 主题的搭建，正在进行进一步的测试工作。如果您在访问本站时遇到问题，请通过邮箱 admin@dimaurora.com 与站长取得联系。

本次更新的具体内容主要包括：

1. 完成了本站各页面的基础布局。

2. 接入了 Twikoo 评论系统。

3. 更新了 Dim Aurora 的社交媒体账号和联系方式。

4. 新增了“关于”页。

5. 完成了基于 GitHub Pages 的部署。

已知问题

在当前版本中，还存在以下已知的问题。

1. 当位于页面最顶端时，页面右上方的 RSS 订阅按钮无法交互。

2. 部分地区（如中国大陆）访问缓慢或无法正常访问。

关于本站的未来发展

Dim Aurora 未来在本站发布的文章，包括但不限于技术分享、资源分享、随笔、生活记录等主题。

本站目前是基于 GitHub Pages 进行部署的。Dim Aurora 正在考察为本站申请中国大陆 ICP 备案，并迁移至中国大陆境内服务器的成本和可行性。这些措施一旦落实，本站在中国大陆访问缓慢或无法正常访问的问题将得到彻底解决。但是，考虑到备案和服务器的成本问题，在未来较长一段时间内，这样的措施可能都将是无法落实的。

本站的前身为 Dim Aurora 基于 Hexo Icarus 主题创建的博客。目前，本站使用的是 Hexo Vivia 主题。Dim Aurora 计划在未来将本站过渡到使用 Hexo Fuwari 主题（2024-03-19 勘误：Fuwari 主题并非基于 Hexo），并最终实现主题框架的全自主化。
